# Libra SDK TypeScript

Library client that can be used to interact with libra network.

## Installation

```bash
npm i @libra/client
```

## Testing

```bash
npm run test
npm run format:check
npm run lint:check
```

## CLI

```bash
./cli.js --help
```

## Usage

```js
import { LibraClient } from '@libra/client';
```
